# Ecoleta
## Projeto desenvolvido na NLW da Rocketseat
O objetivo da aplicação é registrar e pesquisar pontos de coleta de resíduos por cidade.
### Tecnologias utilizadas: Node / Express / sqlite3 / HTML / CSS / Javascript
